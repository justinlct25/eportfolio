What capability is achieved by the flask library?

The flask library is a microframework based on Werkzeug and Jinja2 (Wee, L., 2018) which is a framework for light-weight web application. Using the Flask-RESTful library, which is an extension for Flask, the library enables rapid development of REST API with functions CRUD (Create, Read, Update, Delete).

Using the Flask-RESTful library, this could be one of the distributed systems using distributed computing architecture, which has higher scalability, reliability and is easier to update compared to the monolithic applications.
